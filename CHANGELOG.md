# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-04
### Added
- Initial HVDC Grocery Warehouse Digital Twin Streamlit app.
- Synthetic generators for SKUs, orders, trucks, and warehouse layout.
- Random and PRIME slotting engines.
- Picking travel simulation and heatmap support.
- Scenario comparison view and CSV exports.
- Unit test suite (`unittest`) for core modules.
- GitHub project assets:
  - CI workflow
  - Issue templates
  - PR template
  - Contribution and roadmap docs
